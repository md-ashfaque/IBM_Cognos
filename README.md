# 📊 Student Digital Behavior & Academic Impact Analysis  

> An IBM Cognos Analytics project to explore how digital habits affect student performance and well-being.

---

## 🚀 Project Overview  
This project analyzes student digital behavior—such as screen time, app usage, and lifestyle habits—to understand its impact on academic performance and mental well-being.  

Using **IBM Cognos Analytics**, raw data is transformed into interactive dashboards and reports to uncover patterns in engagement, stress levels, and digital addiction.

---

## 🎯 Objectives  
- Build a structured data model  
- Create interactive dashboards & reports  
- Identify relationships between digital behavior and academics  
- Demonstrate BI skills using Cognos  

---

## 📁 Dataset  
- **File:** `StudentPerformanceFactors.csv`  
- **Format:** CSV  
- **Size:** ~1000 records  
- **Domain:** Student lifestyle & digital usage  

---

## 📌 Features  

- 👤 **Demographics:** age, gender  
- 📱 **Usage:** screen time, social media, gaming  
- 📚 **Productivity:** study/work hours  
- 😴 **Health:** sleep, stress levels  
- 🔔 **Engagement:** notifications, app usage  
- ⚠️ **Behavior:** addiction level  
- 🎓 **Impact:** academic performance  

---

## 📊 Key Insights  

- 📈 Higher screen time → more social media & gaming  
- 🎓 Excessive usage → negative academic impact  
- 😴 Less sleep → higher stress  
- ⚠️ Frequent notifications → higher addiction  
- ✅ Balanced routine → better performance  

---

## 🔮 Predictive Insight  

Students with:  
- High screen time  
- Low sleep  
- Frequent notifications  

➡️ Likely outcomes:  
- Increased addiction  
- Lower academic performance
